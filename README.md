Weekly Planner App

A clean, mobile-friendly task planner with 3 categories:

📚 School — lessons, homework, tests
🏃 Sport — workouts, runs, games
🎮 Life — personal tasks & hobbies
How it works:

Open the app → pick a category on the home screen
Add tasks by typing and pressing +
Tap the circle to check off a task ✓
Hover a task and click the trash icon to delete it
Progress counter shows X/Y done at the top
Key features:

🌍 47 languages supported — switchable anytime via the flag dropdown (with search)
💾 Tasks are saved to the database automatically — they persist across sessions
✨ Smooth animations (Framer Motion) on all interactions
📱 Fully responsive — works great on mobile and desktop
🎨 Gradient color themes per category (blue, orange, purple)
